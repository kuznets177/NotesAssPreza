Шаг 3. Описать модель заметки (Note)
from django.db import models

class Note(models.Model):
    title = models.CharField(max_length=200)       # Заголовок заметки
    content = models.TextField(blank=True)         # Текст (Markdown)
    created_at = models.DateTimeField(auto_now_add=True)  # Дата создания

    def __str__(self):
        return self.title


Шаг 4. Создать и применить миграции
    python manage.py makemigrations notes
    python manage.py migrate

    - makemigrations  создаёт миграционный файл — инструкцию «создать таблицу Note с такими полями» .
    -  migrate  реально применяет эту инструкцию к базе и создаёт таблицу.
